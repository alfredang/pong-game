# Neon Pong: Futuristic Arcade Experience

A sleek, modern Pong game built with Vite, Tailwind CSS (v4), and Vanilla JavaScript.

## üéÆ Features
- **Sleek Neon Aesthetics**: Dark mode UI with cyan/magenta glow effects and a CRT scanline overlay.
- **Advanced AI**: Challenging AI with three difficulty levels (Easy, Normal, Hard), featuring reaction delays and precision scaling.
- **Dynamic Physics**: Ball speed increases with hits and over time.
- **Synthesized Audio**: Retro-futuristic sound effects generated via Web Audio API.
- **Responsive Design**: Scales for desktop and mobile with touch support.
- **Full Game Loop**: Includes Pause, Restart, and a Win Condition (First to 10).
